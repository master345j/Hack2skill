## 🚀 Start Using It

### Option 1: Local Demo (30 seconds)
```
1. Clone: git clone https://github.com/master345j/Hack2skill.git
2. Open: open Hack2skill/frontend/index.html
3. Click: "Continue as Demo User"
```

### Option 2: Full Setup
```bash
git clone https://github.com/master345j/Hack2skill.git
cd Hack2skill

# Terminal 1: Backend
cd backend && npm install && npm start

# Terminal 2: Frontend
cd frontend && python -m http.server 8000
```

### Option 3: Deploy to Production
**Frontend**: Push to Netlify/Vercel from GitHub
**Backend**: Deploy to Heroku/Railway from GitHub
